# Case Studies Tutorials

This directory contains hands-on tutorials for MetaSpliceAI case study workflows.

## Available Tutorials

### [ClinVar Workflow Steps 1-2 Tutorial](CLINVAR_WORKFLOW_STEPS_1_2_TUTORIAL.md)

**Complete hands-on tutorial for ClinVar VCF processing with enhanced coordinate validation**

**What you'll learn:**
- VCF normalization and variant parsing workflows
- Enhanced coordinate system validation using VCF coordinate verifier
- Strand-aware variant verification with gene context
- Genome browser integration for variant investigation
- Comprehensive troubleshooting and validation procedures

**Key Tools Covered:**
- `bcftools` for VCF processing
- Enhanced VCF coordinate verifier with coordinate system validation
- Strand-aware variant analysis with genome browser links
- Contig compatibility testing

**Prerequisites:**
- Basic knowledge of VCF files and genomic coordinates
- Access to MetaSpliceAI installation
- `bcftools` and `pysam` installed

**Time Investment:** ~30-45 minutes for complete walkthrough
